Groovy TestAnswer0r

Randomize your answers

General

Groovy TestAnswer0r is an application for generating random test answers.

Usage

Enter the desired number of questions (Question count) and number of choices (Answer count), select whether the output is in lower case or upper case (Upper case), optionally set a new randomization seed (Randomize) and generate (Generate) your results in the output field. The application automatically saves your settings, so when you start the application again, you can continue from where you left off.

Main Window Description

Menu Bar

File

  • Randomize - Generates a new seed for generating results
  • Generate - Generates test answers to the output field
  • Close - Closes the main window

Options

  • Upper case - Sets whether the results are generated in upper case (ABCD) or lower case (abcd)

Help

  • Help Contents (F1) - Displays this help
  • Licence Agreement - Displays the licence agreement
  • About - Displays information about the application

Window Contents

  • Question count - The desired number of questions, has to be in the limit of 1 to 5000
  • Answer count - The desired number of choices, has to be in the limit of 1 to 26
  • Upper case - Sets whether the results are generated in upper case (ABCD) or lower case (abcd)
  • Randomize - Generates a new seed for generating results
  • Generate - Generates test answers to the output field
  • Output field - Generator output is placed here for you to copy
